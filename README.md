# ShakeME

Scan once. Connect instantly.

## Overview
ShakeME is a mobile-first networking app that replaces traditional business cards with a single QR scan.

Users create a digital profile and generate a QR code. When scanned, it instantly reveals contact details, enabling quick connections via WhatsApp, LinkedIn, or saving directly to contacts.

---

##  Features

- QR-based contact sharing
- Two modes:
  - **Shake Me** → Name, Phone, Role
  - **Hug Me** → Full profile (LinkedIn, Status, Bio)
- One-scan profile access
- WhatsApp direct messaging
- Save contact (vCard download)
- Mobile-first responsive design
- PWA-ready (installable on phone)

---

## How It Works

1. Create your profile
2. Generate your QR code
3. Someone scans it
4. Your profile opens instantly
5. They can:
   - Message you
   - View LinkedIn
   - Save your contact
